# NorthStar Database Analytics Project

This repository contains the implementation files and analytical notebooks developed for the NorthStar Urban Mobility and Logistics database analytics coursework project.

## Repository Contents

* NorthStar_SQL_R.ipynb
  SQL queries and R-based operational analysis

* NorthStar_Analytics.ipynb
  Python data processing, analytics and visualisation

* NorthStar_MongoDBipynb.ipynb
  MongoDB CRUD operations, aggregation pipelines, indexing and query optimisation

## Technologies Used

* R
* SQL (sqldf)
* Python
* Pandas
* NumPy
* Matplotlib
* MongoDB
* PyMongo
* Google Colab
